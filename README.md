# 🩺 Pneumonia Detection using Chest X-Ray Images

An AI-powered deep learning project that classifies chest X-ray images to detect **Pneumonia (Positive/Negative)** using the Pneumonia-MNIST dataset.

---

## 🚀 Project Overview

Pneumonia is a serious lung infection that can be life-threatening if not diagnosed early. This project leverages **Deep Learning** to automatically analyze chest X-ray images and predict whether a patient is infected with pneumonia.

The model is trained on the **Pneumonia-MNIST dataset**, making it lightweight and efficient for experimentation and deployment.

---

## 🎯 Objectives

- Build a binary image classification model  
- Detect pneumonia from chest X-ray images  
- Achieve high accuracy with a lightweight architecture  
- Provide a scalable solution for medical image analysis  

---

## 🧠 Model & Approach

- Image preprocessing and normalization  
- Convolutional Neural Network (CNN) architecture  
- Binary classification (Positive / Negative)  
- Model evaluation using accuracy and loss metrics  

---

## 📊 Dataset

- **Dataset**: Pneumonia-MNIST  
- Source: MedMNIST  
- Contains labeled chest X-ray images for binary classification
